# Portfolio

Personal portfolio website showcasing selected projects, experience and technical focus.

Live site: https://portfolio-git-master-aoga-j.vercel.app

## Overview

This portfolio highlights case studies and projects with an emphasis on:
- Thoughtful architecture.
- Maintainable frontend systems.
- Clear user experience.
- Organizational goals.

It is intended as a technical and professional overview rather than a personal marketing site.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Vercel (deployment)
